# 🎮 마비팟 숙제 관리 사이트 (Mabipot Homework Tracker)

**마비팟 숙제 관리 사이트**는 길드원 또는 파티원들의 주간 콘텐츠(어비스, 레이드) 진행 현황을 한눈에 파악하고 관리할 수 있는 웹 대시보드입니다. Google Sheets를 데이터베이스로 활용하여 별도의 백엔드 서버 없이도 실시간 데이터 동기화가 가능합니다.

---

## ✨ 주요 기능

### 1. 유저 및 캐릭터 관리

* **멀티 유저 지원**: 여러 명의 길드원을 추가하고, 각 유저별로 여러 개의 캐릭터를 생성하여 관리할 수 있습니다.
* **상세 커스터마이징**: 캐릭터별 이름, 직업, 전투력 설정은 물론, 대표 이미지 URL과 체크박스 포인트 컬러까지 개별 설정이 가능합니다.
* **이미지 위치 조절**: 캐릭터 카드의 이미지 세로 출력 위치(Object-position)를 슬라이더로 간편하게 조정할 수 있습니다.

### 2. 진행 현황 추적 (숙제 체크)

* **주간 콘텐츠 분리**: '어비스'와 '레이드' 카테고리로 나누어 관리합니다.
* **실시간 저장**: 체크박스를 클릭하는 즉시 데이터베이스(Google Sheets)와 동기화되어 새로고침 후에도 상태가 유지됩니다.

### 3. 스마트 개요 (Overview) 모드

* **미진행 목록 필터링**: '개요' 탭에서는 현재 숙제를 완료하지 않은 캐릭터들만 모아서 보여줍니다.
* **필터링 시스템**: 어비스/레이드별, 또는 특정 던전별로 미진행 인원을 빠르게 조회할 수 있는 세그먼트 필터를 제공합니다.

### 4. 반응형 레이아웃

* **사이드바 시스템**: 데스크톱에서는 접이식 사이드바를, 모바일에서는 햄버거 메뉴를 통해 쾌적한 UI를 제공합니다.
* **부트스트랩 기반**: Bootstrap 5를 활용하여 다양한 기기 해상도에 최적화된 카드 레이아웃을 구현했습니다.

## 📸 주요 화면 구성

* **개요**: 전체 인원의 미진행 숙제 통계 확인 및 필터링.
* **유저 탭**: 특정 유저의 캐릭터 카드 목록 및 숙제 체크/수정.
* **수정 모드**: 캐릭터 정보 변경 및 삭제 기능을 포함한 모달 팝업.
